# SheDrive Care

Professional mobile auto care service designed for women.
Focused on brake systems and basic maintenance with transparent pricing.

## About

SheDrive Care provides:

- Brake pad replacement
- Brake disc replacement
- Basic maintenance
- Preventive inspections

All parts pricing displayed is referential and based on public retail sources.
Final pricing is confirmed before booking.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- JSON-based vehicle data
- GitHub Pages ready

## Purpose

This project represents the informational website for SheDrive Care,
a women-focused mobile automotive service.
